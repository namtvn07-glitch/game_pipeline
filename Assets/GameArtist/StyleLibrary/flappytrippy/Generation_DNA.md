# I. VISUAL DNA

**Shape Language**
- Soft, rounded, and chubby silhouettes with kawaii/plump proportions.
- Bold chunky forms avoiding sharp, aggressive spikes.
- UI elements and typography must use 3D pixelated retro styling with soft bevels.

**Color Script (Rarity)**
- Highly vibrant, warm, saturated primary colors (neon orange, bright yellow, cyan, hot pink, bright green).
- High contrast against dark/cool environments (e.g., starry space, deep blue/purple backgrounds).
- No desaturated, washed out, or muddy colors. 

**Material Polish**
- Premium plastic/vinyl toy aesthetic with high-gloss specularity and soft gradients.
- Smooth tactile bevels with strong, clean rim lighting.
- Assets should feel highly 3D-rendered but presented as 2D orthographic sprites.

# II. PRODUCT LOGIC

**Readability (3-Second Rule)**
- The primary color and silhouette must instantly communicate the object's function (e.g., Red/Warning = Danger/Explosive, Gold = Points/Coins, Cyan/Glow = Magic/Shield).
- High-visibility contrast is mandatory. The core gameplay elements must distinctly pop off the dark background.

**Progression Logic**
- Basic obstacles remain stationary or solidly painted.
- Complex/rarer obstacles feature glowing/emissive energy states or directional/warning indicators.

**The Juice (Animation states)**
- UI elements must feel like chunky, satisfying tactile push-buttons. 
- Projectiles and powerups require glowing or emissive core states.
- Characters and enemies should appear squishy or plump.

# III. TECHNICAL EXCELLENCE

**Mesh / Topology rules**
- Since this is a 2D workflow, all assets must be rendered with an orthographic/front-facing, slight top-down viewing angle, avoiding intense perspective distortion.
- Edges must be clean with alpha channels fully anti-aliased.

**Texel Density**
- High resolution, aiming for crisp edges suitable for retina mobile screens.
- Keep gradients smooth and avoid pixelation/noise (except in stylized title text). Keep it clean and readable at a thumbnail size.

**Modular design / Recolor limits**
- Blocks and standard obstacles must align to a universal grid format for seamless tileability. 
- Metallic or plastic bases should be hue-shiftable without losing their specular highlighting.
