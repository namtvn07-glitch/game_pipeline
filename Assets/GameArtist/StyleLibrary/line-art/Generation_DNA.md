# Generation DNA: Line-Art

## Core Prompt Injection
Append these visual anchors to all generation prompts requesting this style:
"crisp strict black and white line-art, intricate thin and bold black ink outlines, uncolored, pure white background, flat and 2D appearance, comic book style inking, minimalist shading without gradients, highly detailed contours"

## Negative Prompts
MUST EXCLUDE:
"color, gradients, shading, 3D render, photorealistic, noise, sketchy lines, messy inks, grey tones, background elements, chaotic textures"
