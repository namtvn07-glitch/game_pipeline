# Generation DNA: Line-Art

## Core Prompt Injection
Append these visual anchors to all generation prompts requesting this style:
"crisp contour line-art, intricate thin and bold black ink outlines, vibrant flat colors, pure white background, flat and 2D appearance, comic book style inking, minimalist shading without gradients, highly detailed contours"

## Negative Prompts
MUST EXCLUDE:
"gradients, heavy shading, 3D render, photorealistic, noise, sketchy lines, messy inks, grey tones, background elements, chaotic textures"
