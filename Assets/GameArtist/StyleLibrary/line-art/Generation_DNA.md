# I. VISUAL DNA

**Consistency Rules**
- The style MUST be fully 3D rendered and casual-mobile optimized. NO 2D flat line-art.
- Shapes must be intentionally thick, chunky, and distinctly rounded. Avoid sharp angles or ultra-realistic details.

**Shape Language**
- Soft, pillowy, hyper-rounded forms.
- Friendly, bubbly proportions.
- Toy-like aesthetic with oversized, bold silhouettes.

**Color Script (Rarity)**
- **Standard Rarity**: Vibrant pastel and bold primary colors, highly saturated.
- **Epic/Rare**: Enhanced contrast, introducing metallic touches or high-gloss accents.
- **Overall Palette**: High contrast, warm, playful, and cheerful.

**Material Polish**
- Soft matte plastic, glossy ceramics, or clean soft wood-composites.
- Zero realistic micro-textures (no grunge or scratches).
- Shading relies on clean, soft gradients and smooth specular highlights.

---

# II. PRODUCT LOGIC

**Readability (3-Second Rule)**
- The primary silhouette must instantly convey the object type or character archetype at small mobile thumbnail sizes.

**Progression Logic (Level Upgrades)**
- As objects level up, they should introduce more distinct, smooth geometric segments, or slightly glossier specular highlights without ever adding visual noise.

**The Juice (Animation States)**
- Materials should look tangible and catch light evenly. Ensure designs look good when cleanly squashing and stretching (bouncy physics).

**UI Interaction**
- Assets are perfectly suited for large casual game UI buttons, pop-up distinct icons, or idle game elements. The extreme visual softness should intuitively encourage tapping.

---

# III. TECHNICAL EXCELLENCE

**Mesh / Topology Rules**
- If utilized in 3D, topology should be mobile-optimized focusing on smooth bevels. If 2D sprites, they need perfect, clean alpha cutouts.

**Texel Density**
- Uniform color distribution across regions—no hyper-detailed, noisy patches mixed with large flat planes. It must be consistently stylized.

**Modular Design / Recolor Limits**
- Objects should easily adapt to global hue-shifts for easy seasonal reskins. Ensure base grayscale values support high-saturation uniform color tinting.
